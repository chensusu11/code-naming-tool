# 代码命名小工具
##  由于网络原因，未上传exe文件。如需要直接下载exe文件，可以链接到国内的gitte：https://gitee.com/chensusu11/code_naming_tool/dist中 win-unpacked.zip
一款轻量、易用的中文转代码命名工具，支持一键生成下划线/小驼峰/大驼峰格式，支持单词简写、Enter 键快捷翻译，可直接在 Windows 上运行。

## 📌 功能特点
- 🎯 中文一键转代码命名（下划线/小驼峰/大驼峰）
- ⚡ 输入后按 Enter 键快速翻译，无需点击按钮
- 📝 单词数 >3 时自动简写（每个单词保留前3个字母）
- 📋 生成结果一键复制，直接粘贴到代码中
- 🖥️ 纯桌面端运行，无需浏览器/网络（基础词库离线可用）

## 📥 安装使用
### 方式1：绿色版（免安装）
1. 解压 `dist/win-unpacked.zip` 到任意文件夹
2. 双击文件夹内的 `代码命名小工具.exe` 直接运行
3. （可选）右键创建快捷方式到桌面，方便后续使用

## 🚀 使用教程
1. 打开工具后，在左侧输入框中输入需要转换的中文（支持多行）
   - 示例：输入「用户登录验证」
2. 按下 Enter 键或点击「开始翻译」按钮，自动生成三种格式：
   - 下划线：`user_login_ver`（单词数>3时自动简写）
   - 小驼峰：`userLoginVer`
   - 大驼峰：`UserLoginVer`
3. 点击右侧「复制」按钮，可一键复制对应格式的命名
4. 直接粘贴到代码中使用即可

### 特殊规则
- 当翻译后的单词数量 >3 时，自动对每个单词取前3个字母简写
  - 示例：「数据库连接池配置」→ 下划线格式 `dat_con_poo_con`
- 基础中文词汇（如「帧频」「分页」）使用本地词库，精准度更高
- 未匹配到本地词库的内容，会自动调用翻译接口生成（需联网）

## 🛠️ 常见问题
### Q1：运行 exe 提示「缺少 xxx.dll」？
- 原因：Windows 缺少 VC++ 运行库
- 解决：下载安装 [微软常用运行库合集](https://learn.microsoft.com/zh-CN/cpp/windows/latest-supported-vc-redist?view=msvc-170)（x64 版本）

### Q2：按 Enter 键没反应？
- 确保输入框处于激活状态（点击输入框后再按 Enter）
- 检查输入框是否为空，空内容不会触发翻译

### Q3：生成的命名不符合预期？
- 本地词库仅覆盖常用开发词汇，特殊领域词汇可手动修改生成结果
- 联网状态下翻译结果更精准，建议保持网络畅通

### Q4：打包时未生成安装版 exe？
1. 手动安装 NSIS 工具：https://nsis.sourceforge.io/Download
2. 配置系统环境变量（添加 NSIS 安装路径到 Path）
3. 重新执行 `npm run build` 打包
